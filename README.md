# Exploratory-Data-Analysis-EDA
# Netflix Data Analysis Project

## Project Overview

This project analyzes the Netflix Titles dataset using Python, Pandas, Matplotlib, and Seaborn. The goal is to perform data cleaning, exploratory data analysis (EDA), and create visualizations to understand trends in Netflix content.

## Tasks Performed

* Loaded and inspected the dataset
* Cleaned missing values
* Checked for duplicates
* Performed exploratory data analysis
* Created 6 visualizations
* Generated business insights

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Dataset

Netflix Titles Dataset
